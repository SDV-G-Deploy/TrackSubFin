# TrackSubFin

Простой домашний трекер подписок с автоматической синхронизацией между устройствами.

Демо: **https://sdv-g-deploy.github.io/TrackSubFin/**

## Как пользоваться вдвоём (очень просто)

1. На каждом устройстве откройте приложение и нажмите **«Синхронизация»**.
2. Введите одинаковый **Archive Code** и свой **GitHub Token**, нажмите **«Подключить»**.
3. Готово: дальше синхронизация работает сама:
   - при запуске приложение подтягивает данные из облака,
   - после изменений автоматически сохраняет,
   - каждые ~25 секунд проверяет обновления.
4. Кнопка **«Синхронизировать сейчас»** — только если хотите обновить вручную прямо сейчас.
5. Внизу видно короткий статус:
   - **«Синхронизировано: HH:MM»**
   - или **«Нет связи, повторим автоматически»**

## Что сохраняется

- Подписки хранятся локально в браузере.
- Для общего доступа между устройствами используется GitHub Gist (`tracksubfin-data.json`).

## Локальный запуск

Просто откройте `index.html` в браузере.

## Деплой

При push в `main` GitHub Actions автоматически публикует сайт на GitHub Pages (`.github/workflows/pages.yml`).
