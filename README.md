# TrackSubFin

Лёгкий MVP веб‑приложения для трекинга подписок (без бэкенда, данные в `localStorage`).

## Демо (GitHub Pages)

После деплоя сайт доступен по URL:

**https://sdv-g-deploy.github.io/TrackSubFin/**

## Что умеет

- Добавление подписок с полями:
  - Название продукта
  - Цена
  - Периодичность: `monthly` / `yearly` / `custom-days`
  - Дата следующего списания
  - Карта: название + последние 4 цифры
  - Опционально дата окончания
- Визуализация:
  - Список подписок
  - Горизонтальная timeline по ближайшим списаниям
- Цветовые статусы по ближайшему списанию:
  - `<= 3 дней` — красный + заметное свечение
  - `4–7 дней` — amber
  - `> 7 дней` — нейтрально
- Показ «сколько осталось» до списания и (если есть) окончания.
- Итоговая ежемесячная сумма внизу блока:
  - `yearly -> monthly = price / 12`
  - `custom-days -> monthly = price * (30 / days)`

## Локальный запуск

Откройте `index.html` в браузере.

## Деплой GitHub Pages

В репозитории настроен GitHub Actions workflow: `.github/workflows/pages.yml`.

Он автоматически деплоит статику из ветки `main` на GitHub Pages при каждом push.
